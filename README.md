# Operating Systems Course Assignments  
 
This repository contains all assignments related to the Operating Systems course.  
Each assignment focuses on core OS concepts such as process management, scheduling, memory allocation, deadlocks, file systems, and synchronization.  
The above repository is organized to provide clear access to code, reports, and outputs for each task.
